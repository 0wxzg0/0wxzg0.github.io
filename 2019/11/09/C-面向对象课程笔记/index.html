类与对象

类：class Box{};
对象：Box box;

成员函数的定义：

  • 在类中定义
  • 单独使用范围解析运算符::定义

访问修饰符

成员和类的默认访问修饰符为private

public成员

外部可访问的。
如：直接在主函数里对变量进行赋值

private成员

在类的外部不可访问/查看。
只有类，友元函数可以访问。
一般地，在私有区域定义数据，在共有区域定义访问/修改数据的函数。

protect成员

与私有成员类似。不同在于，保护成员在派生类中是可访问的。

构造函数&析构函数

构造函数

一种特殊的成员函数，在每次创建类的新对象时执行。
可用于为某些成员变量设置初始值。
名称与类相同，不会返回任何类型（包括void）。

输出：
Object is being created

带参数的构造函数

在创建对象时给对象赋初值。

输出：
Object is being created,length=10

用初始化列表初始化字段

析构函数

一种特殊的成员函数，在每次删除所创建的对象时执行。
有助于在跳出程序（关闭文件、释放内存等）前释放资源。
名称与类相同，在前面加~。
不返回任何值，也不能带有任何参数。

输出：

拷贝构造函数//待完善

相关文章
在创建对象时使用同一类中已创建对象来初始化新创建的对象。
常用于：

  • 初始化新创建的对象
  • 复制对象，把它作为参数传递给函数
  • 复制对象，从函数返回这个对象

如果在类中没有定义拷贝构造函数，编译器会自行定义一个。
如果类带有指针变量，并有动态内存分配，则它必须有一个拷贝构造函数。
常见形式：
classname(const classname &obj){ }

在Line b=a中，系统为对象b分配内存，通过拷贝构造函数完成对a的复制。
可见，拷贝构造函数是一种特殊的构造函数。

指针

this指针

每个对象都能使用this指针来访问自己的地址。
由此，在成员函数内部，this指针可以用来指向调用对象。

指向类的指针

与指向结构的指针类似，使用成员访问运算符 ->

友元函数&友元类

友元函数/类有权访问类的私有成员、保护成员。

  • 友元函数定义在类外部（不是成员函数）
  • 声明友元类需在原类定义中声明：friend class classname;

静态成员

静态成员变量

使用static关键字将类成员定义为静态的
静态成员在一个类的所有对象中共享，始终只有一个副本
静态成员的初始化不能放在类的定义中

输出：
2

静态成员函数

静态成员函数只需要使用classname::就能访问（即：在类对象不存在的情况下也能被调用）
静态成员函数只能访问静态成员数据、其它静态成员函数、类外部
静态成员函数无this指针

输出：
0

继承

基类&派生类

基类：已有的类
派生类：继承基类的新建类，拥有基类定义的成员变量与函数
一个类可以从多个基类继承数据和函数。
例如：
class NewClass:public BasedClass

访问控制与继承

访问 public protected private
同一个类 ◯ ◯ ◯
派生类 ◯ ◯ ×
其他类 ◯ × ×
派生类无法继承的：
* 构造函数、析构函数、拷贝构造函数
* 重载运算符
* 友元函数
## 多继承

环状继承->虚继承

如：A->D,B->D,C->A&B
该继承会使D创建两个对象，需改用虚继承模式

重载&模板&虚函数

函数重载

在同一个作用域中可以声明几个功能类似的同名函数，它们的形参必须不同（个数、类型、顺序等）。

运算符重载

运算符重载无法改变：

  • 语法结构
  • 操作数的个数
  • 优先级
  • 结合性

例：重载加号

虚函数

使用virtual声明函数，使类被继承后能修改同名函数的内容
(继承类中不需要定义为virtual)

  • 构造函数不能被定义为虚函数
  • 析构函数可以被定义为虚函数
  • 析构函数定义成虚函数的意义：基类指针可以指向派生类的对象，删除该指针delete []p，会调用该指针指向的派生类的析构函数（如果不声明为虚函数，则调用基类的析构函数）*

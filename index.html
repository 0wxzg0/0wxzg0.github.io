第一章 基础知识

计算机内部是采用什么计数制来表示数和信息的？

  • 二进制B 十进制D 十六进制H 八进制O
  • 二进制计量单位
    • 比特bit
    • 字节byte=8bit
    • 字word=2byte
  • 基本方法：除法 降幂法

计算机中的整数有几种表示方式？最常用的是哪一种码制？

  • 原码：符号+绝对值
    • 0的表示方法不唯一（符号可正可负）
  • 反码：正同原；负数值取反
    • 0的表示方法不唯一（符号可正可负）
  • 补码：正同原；负取反加一
    • 0的表示方法唯一，即00000000B
    • 比原码、反码多表示一个数(-2n ~ 2n-1)

求补的运算规则

  • 十进制：2n-(原数)
  • 二进制：负数取反加一
  • [X-Y]补=[X]补+[-Y]补

  • A-Z：41H-5AH
  • a-z：61H-7AH
  • 回车CR：0DH
  • 换行LF：0AH
  • $：24H
  • 空格SPACE：20H

汇编程序CPU可见的寄存阵列用途（记忆）

CPU组成：

  • 算术逻辑部件ALU
  • 控制逻辑EU
  • 工作寄存器

寄存器（Register）：

  • 处理器中临时数据存储单元
  • 存放运算过程中所需要的或所得到的信息
  • 访问速度：ns级（快于内存μs级）

数据寄存器

  • 4个16位：AX BX CX DX
  • 8个8位：AH AL BH BL CH CL DH DL
  • 4个32位：EAX EBX ECX EDX
  • 专用目的时：
    • AX：乘除指令中存放操作数，I/O指令使用与外设传送数据
    • BX：基址寄存器
    • CX：计数器（移位、循环、串处理指令）
    • DX：双字长运算（与AX组合）存放高位字，I/O操作存放I/O端口地址

指针及变址寄存器

  • 4个16位：SP BP SI DI
  • SP：堆栈指针寄存器
    • 堆栈栈顶偏移量
    • 与SS配合
  • BP：基址指针寄存器
    • 地址偏移量
    • 存放偏移量时，缺省情况与SS配合
  • SI：变址寄存器
    • 串数据的源地址偏移量
    • 存放偏移量时，缺省情况与DS配合
  • DI：变址寄存器
    • 串数据的目的地址偏移量
    • 存放偏移量时，缺省情况与ES配合
  • 缺省=默认

段寄存器

  • 4个16位
  • CS：代码段寄存器
    • 当前运行的程序代码段基地址
  • SS：堆栈段寄存器
    • 堆栈段的基地址
  • DS：数据段寄存器
    • 当前运行程序使用的数据段基地址
  • ES：附加数据段寄存器
    • 当前运行程序使用的附加数据段基地址

控制寄存器

  • IP：指令指针寄存器
    • 代码段中的指令地址偏移量
    • 控制器根据指令字长自动增加
    • 与CS配合，始终指向下一条即将执行的指令的首地址
  • FLAGS：标志寄存器（可能不做要求？）
    • 指：成立1，不成立0
    • 条件码标志：记录程序中运行结果的状态信息，由运算结果自动设置
      • OF 溢出
      • SF 负数符号
      • ZF 零
      • CF 进位/借位
      • AF 辅助进位
      • PF 奇偶
    • 控制标志：通过指令人为设置，控制程序执行
      • DF 方向：用于指示串操作时操作数地址的增减方向（1：SI、DI自动减量）
      • IF 中断：对外部可屏蔽中断请求INTR起作用
      • TF 陷阱：单步运行方式

计算机存储器组织方式

存储单元的地址和内容

  • 存储：以字节为单位，每个字节单元为一个地址
  • 读写：字长16位，占用相继的两字节
    • 低位存入低地址，高位存入高地址
    • 以偶地址访问存储器
    • 字单元地址用其低地址表示
    • 举例：(03080H)=5678H

存储器地址表示方式

  • 一个内存单元地址 = 端基地址:偏移量 = DS:SI

  • 物理地址：每个存储单元的唯一的20位地址

  • 逻辑地址：实模式下，机器指令给出的地址(DS:SI)

  • 段基地址：段起始地址=10H×段寄存器

  • 偏移地址：段内相对于段起始地址的偏移量，又称有效地址EA

    • 基址

      - 寄存器操作数，存储器操作数

    • OUT

      • 长格式：

      • 用于把一种代码转换为另一种

  • 地址传送指令

    • LEA

      • 把源操作数的有效地址EA送到指定的寄存器中

      • 执行两数相减操作DST-SRC后根据结果设置标志位，不改变原值

    • 溢出判断：加减法执行后

      • CF=1，无符号数发生溢出
      • OF=1，有符号数发生溢出

  • 乘法指令

    • MUL
      • 无符号数乘法
      • 目的操作数是累加器AX或AL
      • 2个8位数相乘得16位，2个16位数相乘得32位，存放在DX:AX中
    • IMUL
      • 有符号数乘法

  • 除法指令

    • DIV
      • 源操作数可以用除立即数以外的任意寻址方式
      • 字节除法：AX/SRC8，商AL，余数AH

    • 适用于参数较少，或子程序有多层嵌套、递归调用的情况（变量地址表动态变化）
    • 1.主程序把参数/参数地址压入堆栈
    • 2.子程序使用堆栈中的参数或通过栈中参数地址取到参数
    • 3.子程序返回时使用RET n指令调整SP指针，以便删除堆栈中已用过的参数，保持堆栈平衡，保证程序的正确返回

第九章 BIOS和DOS中断

  • BIOS：固化在ROM中的基本输入输出系统
    • 显示器 INT 10H
    • 键盘 INT 16H
  • DOS：磁盘操作系统，建立在BIOS基础上的PC机操作系统
    • INT 21H
    • AH=1 键盘输入并回显
    • AH=2 显示字符输出
    • AH=9 显示字符串输出
    • AH=0AH 键盘输入到缓冲区
    • AH=4CH 带返回码终止
